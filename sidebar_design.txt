# Sidebar Design Specification

## 1. Theme Variables (CSS)
Defined in index.html

### Light Mode (:root)
- Background: var(--sidebar-bg) -> #f9f8f7
- Border: var(--sidebar-border) -> #e4e4e7
- Surface Hover: var(--surface-l2) -> #f3f2f1

### Dark Mode (.dark)
- Background: var(--sidebar-bg) -> #141414
- Border: var(--sidebar-border) -> #292929
- Surface Hover: var(--surface-l2) -> #292929

### Mobile Dark Mode (@media max-width: 1024px)
- Background: #000000
- Border: #1a1a1a

## 2. Component Structure & Styling (Tailwind CSS)

### Main Container
- Position: fixed inset-y-0 left-0 z-[100]
- Layout: flex flex-col h-full
- Width: w-full lg:w-[320px]
- Appearance: bg-sidebar border-r border-border
- Animation: transform transition-all duration-500 ease-[cubic-bezier(0.2,0,0,1)]
- States:
  - Open: translate-x-0 shadow-2xl
  - Closed: -translate-x-full

### Header Section
- Height: 6rem (h-[6rem])
- Padding: px-6
- Elements:
  - Logo Button: p-1 rounded-xl hover:bg-surface-l2
  - Toggle Button: size-12 rounded-full bg-white (dark:bg-white/10) backdrop-blur-2xl border border-gray-200 (dark:border-white/10) shadow-xl

### Search & Actions Bar
- Container: h-12 mb-6 px-6 relative flex items-center
- Search Input:
  - Base: h-12 px-4 rounded-full bg-white (dark:bg-white/5) backdrop-blur-xl border border-gray-200 (dark:border-white/10)
  - Animation: transition-all duration-500 ease-[cubic-bezier(0.16,1,0.3,1)]
  - Width Logic: 
    - Default: w-[calc(100%-104px)]
    - Focused: w-full z-10
- Action Buttons (Settings & New Chat):
  - Style: size-12 rounded-full bg-white (dark:bg-white/5) backdrop-blur-xl border border-gray-200 (dark:border-white/10) shadow-lg
  - Animation: transition-all duration-300
  - Hide Logic (when search focused): opacity-0 scale-75 translate-x-4 pointer-events-none

### Navigation List (History)
- Container: flex-1 overflow-auto scrollbar-none px-6 space-y-1
- Static Item (History Label):
  - Style: p-3 rounded-2xl hover:bg-surface-l1 hover:text-foreground text-muted-foreground
- Recent Label:
  - Style: text-[10px] text-muted-foreground font-bold uppercase tracking-widest opacity-40 pl-3 py-2
- Conversation Items:
  - Dimensions: w-full h-[52px]
  - Shape: rounded-2xl
  - Active State: bg-surface-l1 text-foreground font-bold
  - Inactive State: text-muted-foreground hover:bg-white/5 hover:text-foreground
  - Delete Button: size-8 rounded-xl hover:bg-red-500/10 hover:text-red-500

## 3. Icons Used
- Logo: Custom Image (Light/Dark variants)
- SearchIcon
- SettingsIcon
- SquarePenIcon (New Chat)
- ChevronsRightIcon (Toggle)
- Trash2Icon (Delete)
- HistoryIcon (SVG Path)
